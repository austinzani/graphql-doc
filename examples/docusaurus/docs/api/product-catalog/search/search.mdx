# search


Search across products, users, and categories

Returns a union type that can be Product, User, or Category.

**Type:** `query`

### Arguments

| Name | Type | Description |
| :--- | :--- | :--- |
| `query` | `String` | Search query |
| `limit` | `Int` | Maximum results |

### Return Type

[[`SearchResult`](#search-result)]
