# product


Get a product by ID or slug

**Type:** `query`

### Arguments

| Name | Type | Description |
| :--- | :--- | :--- |
| `id` | `UUID` | Product ID |
| `slug` | `String` | Product slug |

### Return Type

[`Product`](#product)
A product available for purchase

Products belong to categories and can have multiple variants (size, color, etc.).
Each product tracks inventory and pricing information.

**Circular Reference:** Product -&gt; Category -&gt; Product (products)
**Deep Nesting:** Product -&gt; variants -&gt; inventory -&gt; warehouse -&gt; region
